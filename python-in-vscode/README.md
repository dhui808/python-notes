### Python in VS Code
[Getting Started with Python in VS Code](https://code.visualstudio.com/docs/python/python-tutorial)

```
Download Python 3.13.7 python-3.13.7-amd64.exe
py --version

Install Python extension for Visual Studio Code
	Click on the Extensions icon in the Activity Bar on the side
	Enter python
	Look for entensions by Microsoft:
		Python
		Python Debugger
		Pylance
		Python Environments
	Install
	Also search for "Jupyter" extensions by Microsoft. Install them.

Create Python project structure (workspace)
A sample: https://github.com/pypa/sampleproject

Create a project-specific virtual environment
	Using Venv or Anaconda
	View - Command Palette... (Ctrl+Shift+P)
	Start typing the Python
	Select Python: Create Environment
	Select Venv
	Select the interpreter you installed
	.venv folder is created
Select the new environment as your Python Interpreter
	View - Command Palette... (Ctrl+Shift+P)
	Type Python
	Select Python: Interpreter command
	Select the interpreter under your .venv folder

```
